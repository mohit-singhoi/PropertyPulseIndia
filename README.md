# 📖 Overview

**PropertyPulseIndia** is a Machine Learning-powered web application that predicts residential property prices across **15+ major Indian cities**. The platform allows users to estimate both **Buy/Sell** and **Rental** property prices using real estate features such as location, area, bedrooms, bathrooms, furnishing status, property age, parking availability, and more.

The project combines **Data Science, Machine Learning, and Web Development** into an intuitive application that delivers real-time price predictions through a modern and responsive user interface.

This project demonstrates the complete Machine Learning lifecycle, including data preprocessing, feature engineering, model training, evaluation, and deployment.

# 🚀 Installation

Clone Repository

```bash
git clone https://github.com/mohit-singhoi/PropertyPulseIndia.git
```

Navigate to project

```bash
cd PropertyPulseIndia
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run application

```bash
streamlit run app.py
```
